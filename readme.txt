Project Name: Island Escape
What the project is: A game that can be run and played

Run the project by running the "Island Escape.py" file

Pygame is required to run the project.

Pressing "r" while on a level restarts/resets that level.
Pressing "q" while on a level quits the level and goes back to the level select screen.
Use the konami code on the title screen to automatically unlock all levels.
